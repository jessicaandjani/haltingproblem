Halting Problem

Halting problem adalah salah satu peroslan insolvable yang terkenal. Halting problem dikemukakan oleh Alan Turing pada tahun 1963. Halting problem adalah sebuah persoalan untuk memutuskan apakah program dengan input tertentu akan terus berjalan tak terbatas (infinite loop) atau berhenti (halt).

Contoh Halting Problem

Kode di atas akan membuat program akan terus berjalan (infinite loop ). Salah satu cara untuk membuat program di atas berhenti adalah dengan membuang if yang berada di dalam while.

Program di atas akan berhenti (halt) ketika x mencapai angka 5.

Dalam kasus ini, program yang diberikan merupakan program yang sangat sederhana sehingga dapat mudah untuk menentukan input apa yang menyebabkan program tersebut berhenti. Namun, pada kasus dengan program yang sudah kompleks akan sangat sulit untuk menganalisisnya. Menurut Alan Turing, sangat mustahil ada sebuah metode umum yang bisa menganalisa semua program, untuk memutuskan masukan apa yang dapat menyebabkan program tersebut berhenti (halt).

Cara Menghindari Halting Problem

Untuk menghindari halting problem, setiap program harus dipastikan dapat berhenti atau bahkan berhenti sebelum waktu yang diberikan.
